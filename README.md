# Get EXP From Nature
- The Minecraft mod, NeoForged mc version 1.21.1
- Get 2 exp on every block related to stones break.
- Get 1 exp on every block releted trees break.
- All orb exp works with mending.

- `.\gradlew --refresh-dependencies`
- `.\gradlew clean`
- `.\gradlew runClient`
- `.\gradlew build`
